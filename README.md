This tool allows you to easily place spawns for enemies for the RPG X gamemode.
It can be equipped over the ulx menu -> Utility -> giveplacer
Left click: places a spawn point
Right click: Opens settings menu

Saves to your local game. After finishing copy the saved file to the same location on the server and change the map.
